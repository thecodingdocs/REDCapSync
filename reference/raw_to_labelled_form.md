# Raw to Labelled REDCap forms

Raw to Labelled REDCap forms

## Usage

``` r
raw_to_labelled_form(form, project)
```

## Arguments

- form:

  data.frame of raw REDCap to be converted to labelled REDCap

- project:

  A validated `project` object containing REDCap project data and
  settings. Generated using
  [load_project](https://thecodingdocs.github.io/REDCapSync/reference/setup-load.md)
  or
  [setup_project](https://thecodingdocs.github.io/REDCapSync/reference/setup-load.md)

## Value

project object
